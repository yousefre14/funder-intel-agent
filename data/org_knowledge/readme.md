# Organization Knowledge Base

Put your organization's documents here.
The agent will read ALL files in this folder.

## Recommended Files

1. `mission_and_programs.md` — Your mission, programs, what you do
2. `theory_of_change.md` — How you believe change happens
3. `outcomes_and_impact.md` — Data, results, success stories
4. `grant_voice_examples.md` — Past successful grant language
5. `org_overview.md` — General overview, history, team

## Format

- Use Markdown (.md) or plain text (.txt)
- Be specific — include numbers, quotes, program names
- Write how you ACTUALLY talk about your work
- Include both formal (grant language) and informal descriptions
